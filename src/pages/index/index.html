Пример № 1.

1) Задача. Составить уравнение заданной окружности

Условие:

Составьте уравнение окружности с центром в точке O (2;-3) и радиусом R = 4.

Решение:

Обратимся к формуле уравнения окружности: (x - x0)2 + (y - y0)2 = R2

Подставим значения в формулу:
Радиус окружности R = 4.
Координаты центра окружности (в соответствии с условием):
a = 2
b = -3

Получаем:
(x — 2)2 + (y - (-3))2 = 42
или
(x — 2)2 + (y + 3)2 = 16.

2) Задача. Находится ли точка внутри окружности

Условие:

Проверить, находится ли точка A(2;3) внутри окружности с уравнением (x — 2)2 + (y + 3)2 = 16.

Решение:

Если точка находится внутри данной окружности, то данное выражение должно быть верно:
(x - x0)2 + (y - y0)2 ≤ R2.
В нашем случае:
(x — 2)2 + (y + 3)2 ≤ 16.

В выражение (x — 2)2 + (y + 3)2 ≤ 16 подставим, согласно условию, координаты точки A (2;3), то есть:
x = 2
y = 3

Проверим истинность полученного равенства:
(x — 2)2 + (y + 3)2 ≤ 16
(2 — 2)2 + (3 + 3)2 ≤ 16
0 + 36 ≤ 16
равенство неверно, 36 > 16
Таким образом, заданная точка не находится внутри заданной окружности.

Пример № 2.

1) Задача. Составить уравнение заданной окружности

Условие:

Составьте уравнение окружности с центром в точке O (-1;2) и радиусом R = 4.

Решение:

Обратимся к формуле уравнения окружности: (x - x0)2 + (y - y0)2 = R2

Подставим значения в формулу:
Радиус окружности R = 4.
Координаты центра окружности (в соответствии с условием):
a = -1
b = 2

Получаем:
(x — (-1))2 + (y - 2)2 = 42
или
(x + 1)2 + (y - 2)2 = 16.

2) Задача. Находится ли точка внутри окружности

Условие:

Проверить, находится ли точка A(2;3) внутри окружности с уравнением (x + 1)2 + (y - 2)2 = 16.

Решение:

Если точка находится внутри данной окружности, то данное выражение должно быть верно:
(x - x0)2 + (y - y0)2 ≤ R2.
В нашем случае:
(x + 1)2 + (y - 2)2 ≤ 16.

В выражение (x + 1)2 + (y - 2)2 ≤ 16. подставим, согласно условию, координаты точки A (2;3), то есть:
x = 2
y = 3

Проверим истинность полученного равенства:
(x + 1)2 + (y - 2)2 ≤ 16.
(2 + 1)2 + (3 - 2)2 ≤ 16.
3 + 1 ≤ 16
равенство верно, 4 < 16
Таким образом, заданная точка находится внутри заданной окружности.
